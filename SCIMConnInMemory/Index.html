To build a new connector:
  1. Implement ISCIMConnector.cs
  2. Register your implementation in App_Start\SimpleInjectorWebApiInitilizer.cs
To test your connector:
  1. Start the debugger in Visual Studio
  2. You can use the postman extension in google chrome to make rest calls
  3. There is a sample collection of tests in App_Data\PostmanTests.json
Other files found in App_Data
  1. 10Users.json - a json file with 10 SCIMUsers
  2. Users.json - a json file with thousands of users exported from the MySQL employee database
  3. Groups.json - a json file with SCIMGroups exported from the departments table of the MySQL employee database
  4. employees.mdf - this is the MySQL employee database migrated to SQL server 2012 using the SQL Server Migration Assistant for MySQL
  5. See here for details.
Other notes:
  1. If you have to tweak the JSON serializer settings, you can adjust them in App_Start\WebApiConfig.cs
  2. In order to handle extension data I added an ExtensionData dictionary to SCIMResource.cs
  3. JSON.net will pack any unrecognized fields into this dictionary
